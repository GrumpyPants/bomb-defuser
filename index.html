Welcome to the Bomb Defuser Tools.

This is a simple set of bomb defusal tools to be used for Keep Talking and Nobody Explodes.

This is a work in progress and currently only works for version 1 (verification code 241) of the bomb manual which can be found here.

The Button

Color: White Blue Yellow Red
Label: Abort Hold Detonate
Number of batteries:
Indicators: CAR FRK

Solution:

Blue strip: release when the countdown timer has a 4 in any position.
White strip: release when the countdown timer has a 1 in any position.
Yellow strip: release when the countdown timer has a 5 in any position.
Any other color strip: release when the countdown timer has a 1 in any position.



The Knob

1 = light is on
0 = light is off
For example: 110000 means that only the first two LEDs are on.
Both keys needed in order to guarantee a correct answer.
